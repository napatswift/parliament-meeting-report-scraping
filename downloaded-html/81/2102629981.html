เรื่องที่พิจารณา
ระเบียบวาระกระทู้ถามสด
1. กระทู้ถาม ของ นายเกียรติ สิทธีอมร เรื่อง นโยบายเศรษฐกิจ
2. กระทู้ถาม ของ นายอลงกรณ์ พลบุตร เรื่อง การระบายข้าวของรัฐบาล
3. กระทู้ถาม ของ นายอรรถวิชช์ สุวรรณภักดี เรื่อง ความพร้อมในการป้องกันและแก้ไขปัญหาน้ำท่วม
4. กระทู้ถาม ของ นายจิรายุ ห่วงทรัพย์ เรื่อง การบริหารจัดการน้ำของรัฐบาลในการป้องกันน้ำท่วม
 
ระเบียบวาระกระทู้ถามทั่วไป
1. กระทู้ถาม ของ นายนิยม วรปัญญา เรื่อง ขอให้รัฐบาลแก้ไขปัญหาความเดือดร้อนของราษฎร
โดยปรับปรุงถนนโครงข่ายเชื่อมโยงภาคต่างๆ และขุดลอกแม่น้ำต่างๆ  เพื่อป้องกันแก้ไขปัญหาภัยแล้ง น้ำท่วม
2. กระทู้ถาม ของ นางสาวชะวรลัทธิ์ ชินธรรมมิตร เรื่อง วิกฤตปัญหาภาษาไทยในปัจจุบัน
3. กระทู้ถาม ของ นายนิยม เวชกามา เรื่อง การพัฒนาขุดลอกลำน้ำพุงสกลนคร
4. กระทู้ถาม ของ นายวิวัฒน์ชัย โหตระไวศยะ เรื่อง ปัญหาความเดือดร้อนด้านการเดินทางสัญจรของประชาชนในชนบท
  
ระเบียบวาระเรื่องที่ที่ประชุมเห็นชอบให้เลื่อนขึ้นมาพิจารณาก่อน
ญัตติ เรื่อง ขอให้สภาผู้แทนราษฎรตั้งคณะกรรมาธิการวิสามัญพิจารณาศึกษาและหามาตราการแก้ไขปัญหาราคายางพาราตกต่ำ
